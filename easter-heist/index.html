How it works

  1. Solve the clue and enter your answer.
  2. Travel to the next stop and tap Check My Location.
  3. If GPS is unreliable, use the fallback code for that stop.
  4. Optionally save a proof photo (stored on your device only).

For parents / setup

Search this file for const MISSION =. Edit this block to update:

  • Mission title: MISSION.title
  • Intro text: MISSION.intro
  • Finish text: MISSION.finishMessage
  • GPS radius: MISSION.gpsRadiusMeters
  • Clues, answers, coordinates, fallback codes: each entry in MISSION.stops

Progress and proof photos are saved to localStorage in this browser only. No backend and no photo upload.
